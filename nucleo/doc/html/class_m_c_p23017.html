Detailed Description

Hardware class for the MCP23017 GPIO Extender PortA is for The LEDs
PortB is for the hal sensor

PortB pins 1-8
PortA pins 21-28
VDD pin 9
VSS pin 10
SCL pin 12
SDA pin 13
A0,A1,A2 pins 15,16,17
